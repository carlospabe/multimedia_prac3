Página de información turística de Jurassic Park.

- **Home**

- **About**
  - Info general
  - Localización, mapa, etc.
  - Contacto
- **Instalaciones**
  - Actividades y puntos de interés
  - Hoteles
  - Restaurantes
- **Dinosaurios**
  - Info de dinosaurios con imágenes y texto plano
